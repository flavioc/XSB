Download the Executables

  • Windows 95/NT [xsb.zip]

    xsb.zip is a WinZip file, which includes xsb.exe (without ODBC interface), xsb_odbc.exe (with ODBC interface), and other necessary library files. It was compiled in Microsoft Visual C++ 4.1.

    How to use it:

    • Use WinZip to extract all the files in xsb.zip (make sure the Use Directory Names check box is checked, and the Extract To directory is set to the place where you want to put the XSB system (for example, "c:"));
    • If you set the Extract To directory as "c:", then the XSB system will be installed in "c:\xsb", which contains four subdirectories: cmplib,emu,lib, and syslib.
    • The executable file is c:\xsb\emu\xsb.exe. To run it you need to specify the path for the libraries -- you can type in a DOS terminal:

      C:\>\xsb\emu\xsb.exe -i -D \xsb

  • Windows 3.1 and DOS:

    The executables of version 1.8 or later for Windows 3.1 and DOS are still not available. Here is an old version (1.7.2): xsbdos.zip (installation instructions can be found in the XSB Manual). Please come back later.
