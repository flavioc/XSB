Download the System

You can download the zipped file of the XSB system by clicking over one of the following links:
  • XSB.tar.gz
    This file includes all source code, examples, manual1.ps, and manual2.ps.

  • XSB.tar.Z
    This file includes all source code, examples, and manual1.ps, manual2.ps)

  • XSBwind.zip
    This is a WinZip file for Windows 95/98/NT, which includes source code, examples, manual1.ps, manual2.ps, xsb.exe, and xsb.dll (both xsb.exe and xsb.dll are in the directory $XSB/config/x86-pc-windows/bin/). If you need to build XSB executables with other options, read the manual on how to do this. Note: if you have Cygnus Software's Cygwin and are planning to use it (recommended), then you should download the Unix version of XSB.
